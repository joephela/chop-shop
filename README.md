## Overview

**Chop Shop** is a high-performance monorepo designed for building and testing modern, visually stunning web interfaces. It leverages a robust monorepo architecture to manage multiple packages and applications with ease.

### Features

- **Monorepo Magic**: Powered by Vite and TypeScript for a fast, unified development experience.
- **Premium Components**: A collection of state-of-the-art UI components (like the `RadialButton` and `IconButton`).
- **Dynamic Styling**: Comprehensive support for Tailwind CSS with custom themes (Berry, Ocean, Forest, Volcano).
- **Interactive Playground**: A dedicated sandbox application for rapid prototyping and component verification.

---

## Workspace Structure

- `apps/playground`: The central hub for testing components in real-world scenarios.
- `packages/buttons`: Core interactive elements with advanced physics and screen-aware positioning.

---

## Getting Started

### Prerequisites

- Node.js 20+
- npm

### Installation

```bash
npm install
```

### Local Development

To start the playground and see the components in action:

```bash
npm run dev
```

Check out the `apps/playground` for live examples of:
